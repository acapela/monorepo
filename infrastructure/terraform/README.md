# terraform

### Install terraform

```sh
brew install terraform
```

### Initialize the terraform project

```sh
terraform init
```

### Apply the terraform configuration

```sh
terraform apply
```

### View the terraform output

```sh
terraform output
```

**Example output:**

```
redis_endpoints = {
  "production" = "10.57.83.27"
  "staging" = "10.184.49.219"
}
```
